1 Flower #permalink

The Flower component is a responsive flower, with the following states:

  • mobile (the default)
  • desktop (screen sizes larger than 60em)
  • hover/focus
  • is-pollinating (state triggered by JavaScript)

To view all of these state variations, the user will need to change the viewport width and hover over the component.
